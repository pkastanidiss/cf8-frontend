# Student Notes Platform (CF8 Frontend)

Μια σύγχρονη εφαρμογή ανταλλαγής σημειώσεων για φοιτητές, χτισμένη με **Angular 19** και **Angular Material**.

## Τεχνολογίες
* **Angular 19+**: Framework για το UI.
* **Angular Material**: Για τα UI components (Cards, Forms, Selects).
* **Reactive Forms**: Για τη διαχείριση της εγγραφής και των σημειώσεων.
* **RxJS**: Για τη διαχείριση των HTTP κλήσεων και των streams.

## Βασικές Λειτουργίες
* **User Authentication**: Σύνδεση και εγγραφή με validation (Unique email, password match).
* **Notes Management**: Δημιουργία, προβολή, επεξεργασία και διαγραφή σημειώσεων.
* **Categorization**: Φιλτράρισμα σημειώσεων ανά κατηγορία (π.χ. Προσωπικά, Ιατρική).
* **Responsive Design**: Πλήρως προσαρμοσμένο για desktop και mobile.

## Εγκατάσταση & Εκτέλεση
1. `npm install`
2. `ng serve`
3. Ανοίξτε το `http://localhost:4200`
