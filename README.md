# Student Notes Platform (CF8 Frontend)

Μια σύγχρονη εφαρμογή ανταλλαγής σημειώσεων για φοιτητές, χτισμένη με **Angular 19** και **Angular Material**.

## Σχετικά με το Project
Αυτό το frontend αποτελεί το περιβάλλον διεπαφής για την πλατφόρμα ανταλλαγής σημειώσεων. Επιτρέπει στους χρήστες να:
* **Βλέπουν σημειώσεις** σε δυναμικές κάρτες (Masonry layout).
* **Διαχειρίζονται τις σημειώσεις τους** (Create, Edit, Delete).
* **Πλοηγούνται εύκολα** μέσω ενός διαδραστικού Navbar και Sidebar με animations.
* **Συνδέονται με ασφάλεια** στο λογαριασμό τους.

## Τεχνολογίες
* **Framework:** Angular 19+
* **UI Library:** Angular Material
* **Testing:** Vitest
* **Styles:** Custom CSS3 Animations & Flexbox
